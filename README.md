# 📚 Web Development Course Syllabus

A simple and responsive web application that provides a structured syllabus for a Web Development course. The project is designed to help students easily navigate through different course units using a clean and user-friendly interface.

## 🚀 Live Demo

🔗 https://faisalkhan-fk.github.io/Syllabus-content/

## ✨ Features

- 📖 Course Syllabus Overview
- 📚 Unit-wise Navigation
- 🏠 Home Page
- 💻 Simple and Clean User Interface
- 📱 Responsive Design
- ⚡ Fast Loading

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Git & GitHub
- GitHub Pages

## 📂 Project Structure

```
Syllabus-content/
│── index.html
│── css/
│── js/
│── images/
│── README.md
```

## 📚 Course Content

- Unit 1 – Introduction to HTML & GitHub
- Unit 2 – CSS and CSS Frameworks
- Unit 3 – JavaScript
- Unit 4 – ECMAScript (ES6)
- Unit 5 – Introduction to React JS

## ⚙️ Installation

1. Clone the repository

```bash
git clone https://github.com/faisalkhan-fk/Syllabus-content.git
```

2. Open the project folder.

3. Run `index.html` in your browser.
